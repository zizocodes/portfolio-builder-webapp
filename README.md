# 🎨 Portfolio Builder App

A modern drag-and-drop portfolio website generator built using HTML, CSS, and JavaScript.

This tool allows creatives and developers to create their own portfolio websites without writing any code.

## 🚀 Features

- Drag and drop portfolio sections
- Live preview editor
- Editable text content
- Theme customization panel
- Multiple starter templates
- Dark / Light mode toggle
- Auto-save using localStorage
- Responsive preview (desktop / tablet / mobile)
- Export portfolio as HTML file

## 🛠 Tech Stack

HTML
CSS
JavaScript
LocalStorage API
Drag & Drop API
